# Dashboards & Interactive Outputs

This folder contains the reporting and interactive outputs for the Stroke Supervision & Tracking Database project, including a Power BI dashboard and a Microsoft Access application used for data entry and query-based reporting.

## Power BI Dashboard (`powerbi/`)
- `OHMC_dashboard.pbix` – Power BI source file
- `powerbi_dashboard_overview.jpg` – screenshot of the final dashboard

The Power BI dashboard summarizes stroke encounters and guideline compliance trends for quality improvement and leadership reporting.

## Microsoft Access Interactive App (`access_app/`)
- `OHMC_stroke_tracking_app.accdb` – interactive Access front-end used for structured data entry, query execution, and report-style outputs
- Screenshots include the main menu, patient form, and compliance/non-compliance query outputs

This Access app functions as an interactive dashboard for operational use and supports the reporting pipeline feeding the Power BI visualization layer.
