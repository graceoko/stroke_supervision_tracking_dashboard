# Project Images

This folder contains visual artifacts supporting the Stroke Supervision & Tracking Database project.

## Included Images
- `erd_diagram.jpg` – entity relationship diagram illustrating the database schema
- `data_dictionary.jpg` – overview of key tables and attributes
- `ssms_database_overview.jpg` – SQL Server environment used for database development
- `project_team.jpg` – final project presentation team photo

These images are referenced throughout the repository to provide visual context for system design, implementation, and collaboration.
