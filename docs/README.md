# Project Documentation

This folder contains formal documentation and presentation materials for the Stroke Supervision & Tracking Database project.

## Documents Included
- `final_project_paper.pdf` – comprehensive project documentation covering system design, database architecture, business rules, and limitations
- `final_presentation.pdf` – summary presentation highlighting the problem context, system implementation, and dashboard outcomes

These materials complement the technical artifacts in the main repository and provide additional context for system design and communication.
